# Project Web

Web project developed for academic purposes.

## Description
This project is a web application developed using PHP, HTML, CSS, and MySQL.

The purpose of the project is to demonstrate basic web development concepts, including:
- Front-end interface
- Back-end functionality
- Database connection
- CRUD operations

## Project Structure

- `htdocs/` → main website source code
- `db/` → database files / SQL export

## Technologies Used

- PHP
- HTML
- CSS
- MySQL
- XAMPP

## Installation

1. Clone the repository
2. Move project folder into XAMPP `htdocs`
3. Import the database from `db/`
4. Start Apache and MySQL from XAMPP
5. Open the project in browser
