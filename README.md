# Project Web

A web application developed for academic purposes using PHP, HTML, CSS, and MySQL.

## Description

This project is a web-based application created as part of an academic assignment.  
It demonstrates the implementation of a complete dynamic website with front-end and back-end functionality, connected to a MySQL database.

The project includes:
- User interface pages
- Server-side functionality with PHP
- Database connection and management
- CRUD operations (Create, Read, Update, Delete)

## Project Structure

- `htdocs/` → main website source code
- `db/` → database files / SQL export

## Technologies Used

- PHP
- HTML
- CSS
- MySQL
- XAMPP

## Installation

1. Clone or download the repository
2. Move the project folder into the XAMPP `htdocs` directory
3. Import the database from the `db/` folder into phpMyAdmin
4. Start Apache and MySQL from XAMPP
5. Open the project in your browser

## Notes

This project was developed for educational purposes and is intended for local use with XAMPP.
